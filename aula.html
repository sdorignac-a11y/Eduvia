Entendamos las fracciones

Vamos a trabajar con una explicación clara, simple y visual para entender cómo funciona el tema.

Escribiendo en el pizarrón

Pasos de la explicación

1. Una fracción representa partes de un todo.
2. El número de arriba es el numerador.
3. El de abajo es el denominador.
4. Para operar, primero miramos si tienen el mismo denominador.

Ejemplo guiado

1/2 + 1/4. Llevamos ambas fracciones al mismo denominador: 1/2 = 2/4. Entonces 2/4 + 1/4 = 3/4.

Objetivo de la clase

Que el alumno entienda cómo leer, interpretar y operar fracciones simples sin memorizar mecánicamente.

Mini ejercicio

Calculá: 3/4 - 1/4. Después explicá por qué no tuviste que cambiar el denominador.

Tipo de clase

Seleccioná la estructura base que más se adapte a lo que querés generar.

C
Clase explicativa

Ideal para desarrollar un tema con introducción, conceptos y cierre claro.

R
Clase resumida

Versión más directa para estudiar rápido o repasar antes de una evaluación.

E
Clase con ejercicios

Combina explicación con actividades para practicar dentro del mismo material.

P
Clase para presentación

Pensada para luego pasarla fácil a diapositivas con mejor orden visual.

Datos de la clase

Completá solo lo importante. El panel evita relleno y deja el foco en la acción.

Podés aclarar qué querés priorizar: resumen, profundidad, ejemplos, ejercicios o tono más formal.

Enfoque extra

Activá características para adaptar mejor la salida final.

Esta estructura está pensada para que después puedas reutilizar exactamente la misma lógica visual en documento, presentación y demás paneles de Eduvia.

Vista rápida

Así se ordenaría la clase una vez generada, manteniendo una estructura más limpia y útil.

  • 1
    Introducción clara Presentación breve del tema y del contexto para entrar más rápido en contenido.
  • 2
    Desarrollo principal Explicación ordenada por ideas y subtemas, sin bloques pesados ni confusos.
  • 3
    Cierre útil Resumen corto para repasar o pasar luego a documento o presentación.
